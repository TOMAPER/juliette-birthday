---
name: Scratch card should show photo big before transitioning
description: After scratching, the revealed photo must appear large/fullscreen on screen before any text or next button appears. Don't skip directly to the joke phase.
type: feedback
---

When a scratch card is fully scratched, show the revealed photo prominently (large, centered) on the screen BEFORE transitioning to any joke text or next step. The photo reveal IS the moment — don't rush past it.

**Why:** The user wants the photo to be the punchline. If it shifts directly to text, the joke doesn't land.

**How to apply:** After scratch threshold is reached, transition to a phase showing the photo big with the next button below it. Only then transition to the joke.
